# Interactive-Color-Palette-Generator
Gerador de paletas de cores moderno e interativo desenvolvido com HTML5, CSS e JavaScript puro.

---

# Demo
Projeto Online:
https://www.paleta-cores.netlify.app/

---

# Funcionalidades
* Geração aleatória de paletas de cores;
* Cópia rápida do código HEX;
* Interface moderna e responsiva;
* Feedback visual ao copiar cores;
* Layout adaptável para mobile;
* Design minimalista com gradientes;
* Histórico das últimas 5 paletas geradas;
* Clique em paletas do histórico para restaurar cores;
* Modo Dark/Light;
* Geração de paletas utilizando a tecla `SPACE`;
* Compartilhamento de paletas via URL;
* Compatibilidade com compartilhamento nativo em dispositivos mobile;
* URLs dinâmicas para carregar paletas automaticamente;
* Animações suaves e microinterações visuais.

---

# Tecnologias Utilizadas
* HTML;
* CSS;
* JavaScript.

---

# Como Funciona o Compartilhamento
Cada paleta gerada cria automaticamente uma URL única:
https://paleta-cores.netlify.app/?colors=FF0000-00FF00-0000FF-FFFF00-000000

Ao abrir o link:
* a paleta é restaurada automaticamente;
* as cores são carregadas instantaneamente;
* o usuário pode compartilhar ou salvar combinações específicas.
